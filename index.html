Python 3 and Python 2

Python 2 made code development process easier than earlier versions. It implemented technical details of Python Enhancement Proposal (PEP). Python 2.7 (last version in 2.x ) is no longer under development and in 2020 has been discontinued.

On December 2008, Python released version 3.0. This version was mainly released to fix problems which exist in Python 2. The nature of these change is such that Python 3 was incompatible with Python 2. It is backward incompatible Some features of Python 3 have been backported to Python 2.x versions to make the migration process easy in Python 3.

Python 2.7 has been discontinued as of July3rd 2020 and you should therefore use Python 3.9

Variables

Variables are nothing but reserved memory locations to store values. It means that when you create a variable, you reserve some space in the memory.

Based on the data type of a variable, the interpreter allocates memory and decides what can be stored in the reserved memory. Therefore, by assigning different data types to the variables, you can store integers, decimals or characters in these variables.

Declaring Variables

Python variables do not need explicit declaration to reserve memory space. The declaration happens automatically when you assign a value to a variable. The equal sign (=) is used to assign values to variables. The operand to the left of the '=' operator is the name of the variable and the operand to the right of the '=' operator is the value stored in the variable.

number = 10 #Assigns an integer 10 to number
name = "John" #Assigns a string 'John' to name

Constants

A constant is a type of variable whose value cannot be changed. It is helpful to think of constants as containers that hold information which cannot be changed later.

PI = 3.14 #Assigns 3.14 to constant PI

If else statements

Use the if statement to execute a statement if a logical condition is true (See the logical conditions reference the full list). Use the optional else clause to execute a statement if the condition is false. An if statement looks as follows:

money = 5.00 price = 3.00

if price > money:
print("The price is too expensive")
else:
print("The price is cheap")

The above block of code shows a program which will print a statement depending of the vaule of the variable 'price'. If the value of price satisfies the first condition then it's print statement will be executed and the program will end. However if this condition is not satisfied then the 'else' code block will be executed instead.

We can also add adtional conditions to make our code account for different scenarios using the 'elif' statement. For example we can add a condition which checks to see if the variable 'money' and 'price' have the same value.

money = 5.00
price = 3.00

if price > money:
print("The price is too expensive")
elif price == money:
print("The price is just right")
else:
print("The price is cheap")

The code above is just a modification of the previous example. However this example has added an additional condition which will account for the variables being equal and therefore making the program more robust.

While Loops

A while statement executes its statements as long as a specified condition evaluates to true. If the condition becomes false, statement within the loop stops executing and control passes to the statement following the loop.

The condition test occurs before statement in the loop is executed. If the condition returns true, statement is executed and the condition is tested again. If the condition returns false, execution stops and control is passed to the statement following while.

i = 0

whiile i < 6:
print(i)
i = i + 1

The code above assigns the value 0 to the variable 'i' then creates a while loop which will print it's value and increment it as long as 'i' is less than 6. When 'i' finally becomes 6 then the program will exit the while loop.

Note the while loop will run endlessly if the condition is never met in this case if 'i' never increments above 6.

Functions

A function is a block of organized, reusable code that is used to perform a single, related action when called. Functions provide better modularity for your application and a high degree of code reusing.

def square(x):
""""Returns the square of a given variable"""
return x*x

The above block of code shows the basic layout of a python function. It first defines the function with 'def' followed by the function name and its parameter. It then provides a brief description of the function purporse and then the block of code to be executed. When called a value will be passed into the function for example 2 and its square will be returned which will be 4.
